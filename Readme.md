
## About the project

Example code as of 05.04.2025. (https://chatbot.kz/)

This repository contains example code for a backend service designed to process incoming requests from a WhatsApp bot. The service receives messages and commands from users, processes them, and performs corresponding actions according to the business logic.

The main logic for interacting with the CRM system is implemented in the `crm` application. It handles client data management, records, and synchronizes data between the CRM and the bot.


## О проекте

Пример кода на 05.04.2025. (https://chatbot.kz/)

Данный репозиторий содержит пример кода для бэкенд-сервиса, предназначенного для обработки входящих запросов от WhatsApp-бота. Сервис принимает сообщения и команды от пользователей, обрабатывает их и выполняет соответствующие действия согласно бизнес-логике.

Основная логика взаимодействия с CRM-системой реализована в приложении `crm`. Там управление клиентскими данными, записями. Синхронизация данных из CRM с ботом.
